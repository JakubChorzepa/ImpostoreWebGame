# Impostor Game

A browser-based social deduction party game for in-person play.

## Setup

```bash
# Use correct Node version
nvm use

# Install all dependencies
yarn
yarn install:all

# Start development servers
yarn dev
```

## Access

- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:3001

## How to Play

1. Open the game on multiple devices (same network)
2. One player creates a lobby and shares the code
3. Others join using the lobby code
4. Host starts the game when ready (min 3 players)
5. Each player sees their secret role privately
6. Discuss in real life to find the Impostor!
7. Vote on who you think is the Impostor
8. Results reveal the winner
